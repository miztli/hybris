AccountController Resource

Controller for create, read and update operations on subscription accounts.

POST /sub/cisSubscriptionMock/accounts/

Creates new account. Example URL: POST https://localhost:8443/hybris-cis-mock-subscription-web/sub/cisSubscriptionMock/accounts

Request Body
media type data type description
application/json CisSubscriptionProfileRequest (JSON) create new account request
application/xml cisSubscriptionProfileRequest (XML)
Response Body
media type data type description
application/xml cisSubscriptionProfileResult (XML) the response with new account
