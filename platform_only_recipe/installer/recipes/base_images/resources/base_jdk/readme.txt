To be able to use this image, please execute the following steps:
################################################################

1) Copy your sapjvm package (linux, rpm) into the /java folder
2) Customize the Dockerifle if necessary
3) execute your recipe via installer
4) build image, e.g. with generated ./buildimages.sh 

############################################################

Use Sap jvm Linux Version (to be able to work on centos).
The current default base os is centos:7.
