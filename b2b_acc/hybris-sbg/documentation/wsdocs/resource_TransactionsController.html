TransactionsController Resource

Controller for processing one time transactions.

POST /sub/cisSubscriptionMock/transactions/

Authorizes and captures a one time transaction. Example URL: POST https://localhost:8443/hybris-cis-mock-subscription-web/sub/cisSubscriptionMock/transactions

Request Body
media type data type description
application/json CisSubscriptionPayNowRequest (JSON) the request for processing a one time transaction
application/xml cisSubscriptionPayNowRequest (XML)
Response Body
media type data type description
application/xml cisSubscriptionTransactionResult (XML) the subscription transaction result with details of the AuthCapture
