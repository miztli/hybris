PaymentMethodController Resource

Controller for updating of payment methods.

PUT /sub/cisSubscriptionMock/paymentmethods/{paymentMethodId}

Updates an existing payment method. Example URL: PUT https://localhost:8443/hybris-cis-mock-subscription-web/sub/cisSubscriptionMock /paymentmethods/25562724555

Request Parameters
name type description constraints
paymentMethodId path the id of payment method to update required
Request Body
media type data type description
application/json CisPaymentMethodUpdateRequest (JSON) update payment method request
application/xml cisPaymentMethodUpdateRequest (XML)
Response Body
media type data type description
application/xml cisResult (XML) the response with the updated payment method
