WebSessionController Resource

Controller which manages web sessions with third party billing providers.

POST /sub/cisSubscriptionMock/websessions/

Initializes a session with the third party billing provider. Example URL: POST https://localhost:8443/hybris-cis-mock-subscription-web/sub/cisSubscriptionMock/websessions

Request Body
media type data type description
application/json CisSubscriptionSessionInitRequest (JSON) init session request
application/xml cisSubscriptionSessionInitRequest (XML)
Response Body
media type data type description
application/xml cisSubscriptionTransactionResult (XML) the subscription transaction result with the transaction details
